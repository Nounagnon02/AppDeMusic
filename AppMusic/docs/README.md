# MaestroSpirit - Application Musicale

## Structure du Projet

```
AppMusic/
├── assets/
│   ├── css/
│   │   └── main.css
│   ├── js/

│   └── images/
├── pages/
│   ├── voice.html
```

## Technologies

- HTML5
- CSS3 (Variables CSS, Grid, Flexbox)
- JavaScript ES6+
- LocalStorage pour la persistance

## Installation

1. Ouvrir `index.html` dans un navigateur
2. Aucune dépendance externe requise
